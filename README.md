# 📊 Análisis de Evasión de Clientes (Churn)

## 📌 Descripción

Este proyecto analiza la evasión de clientes (Churn) en una empresa de telecomunicaciones, con el objetivo de entender por qué algunos clientes cancelan el servicio.

## 🎯 Objetivo

- Identificar cuántos clientes abandonan.
- Analizar qué factores pueden influir en la cancelación.
- Obtener conclusiones que ayuden a mejorar la retención.

## 🛠️ Herramientas utilizadas

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📂 Contenido

- Limpieza de datos
- Análisis descriptivo
- Gráficos de evasión
- Conclusiones finales

## ▶️ Cómo usarlo

Abrir el archivo del notebook y ejecutar las celdas en orden.
